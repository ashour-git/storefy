# 5. Accessibility: skip-to-content, aria-labels, keyboard nav

**Label:** ready-for-agent, accessibility

## What to build

Add skip-to-content link, proper `aria-label` attributes on interactive elements, keyboard navigation support, and focus traps for modals/drawers. Meets WCAG 2.1 AA.

## Acceptance criteria

- [ ] Skip-to-content link visible on Tab press
- [ ] All buttons/links have accessible names
- [ ] Cart drawer traps focus when open
- [ ] Arrow keys navigate dropdowns/menus

## Blocked by

None - can start immediately
